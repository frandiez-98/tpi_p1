## **Descripción**



##### Este proyecto consiste en un sistema desarrollado en Python que permite administrar información de distintos países mediante una interfaz de consola. Los datos se almacenan en un archivo CSV, lo que permite conservar los cambios realizados entre distintas ejecuciones del programa.





## **Funcionalidades**



##### El programa permite:

##### 

* ##### Mostrar el listado de países.
* ##### Agregar nuevos países.
* ##### Modificar la información de un país existente.
* ##### Eliminar países.
* ##### Buscar países por nombre (coincidencia parcial o exacta).
* ##### Filtrar países por continente, rango de población o rango de superficie.
* ##### Ordenar países por nombre, población o superficie.
* ##### Mostrar estadísticas generales.
* ##### Guardar automáticamente los cambios realizados en el archivo CSV.
